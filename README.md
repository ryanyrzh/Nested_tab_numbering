# Nested tab numbering

This is a Firefox add-on for [Tree Style Tab](https://addons.mozilla.org/firefox/addon/tree-style-tab/) that overlays tab numbers on the corners of each tab, with two possible counting rules:

- **Nested count.** Hierarchical numbering that follows the tree structure
- **Flat count.** Simple sequential numbering across the window

Right now it includes some simple customization options for text location, font style, etc.

**Compatible with dynamic themes like that offered by [Adaptive Tab Bar Color](https://addons.mozilla.org/en-US/firefox/addon/adaptive-tab-bar-colour/)!**
